### Architecture Notes

- **Shared PI grid**: `BindingDetector._compute_shared_ranges()` finds union birth/persistence ranges across all 3 diagram sets, then passes explicit ranges to `to_image()`. Without this, PI subtraction is meaningless.
- **Significance test optimization**: `_compute_surrogate_score()` reuses cached marginal X images (X doesn't change across surrogates). Reduced 19-surrogate test from >17min to ~13s.
- **Benchmark methods are self-contained**: TE uses histogram-based estimation (no pyinform dep), PAC uses raw Hilbert (no bandpass), CRQA is manual (no pyrqa dep). All work on any 1D time series.
- **CouplingBenchmark.sweep()** handles transient discard, multi-dimensional generator output (auto-extracts column 0), and per-method normalization (rank/minmax/zscore/none).
